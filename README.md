Моя сборка для верстки сайтов
=======

Подключено:
```
Bootstrap 3.0.3
Jquery 2.1.1
Modernizr 2.8.3
```

Установка linux Mint/ Ubuntu

```
git clone git@github.com:nikolaevav/static-page.git project
cd project
bundle
```

Запуск сервера:

```
middleman
```

Открытие в браузере:

```
http://localhost:4567
```


Генерация сайта:

```
bundle exec middleman build
```
